# BC-Static-Website


### CodeMirror Python/HTML,CSS

1. Include JS files in header

```html
<!-- Include for HTML -->
<script src=mode/htmlmixed/htmlmixed.js></script>
<!-- Include for Python -->
<script src=mode/python/python.js></script>
```

2. Script Configure at the end of the body tag

```javascript
// For HTML
mode: 'text/html',
//For Python
mode: 'python',

```
